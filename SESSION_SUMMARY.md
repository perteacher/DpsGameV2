- **신규 사이트**: `https://dnjsrmstm1.github.io/DpsGameV2/`
- **사유**: 기존 DpsGame 레포에서 GitHub Pages Action이 Queued 상태로 무한 대기 (GitHub 서버 이슈). 새 레포로 이전.
- **app.json**: `experiments.baseUrl = "/DpsGameV2/"` 로 변경됨

### 배포 명령 (DpsGameV2)
```powershell
cd C:\Users\Administrator\DpsGame
npx expo export --platform web
git push neworigin master   # neworigin = DpsGameV2 레포
npx gh-pages -d dist -r "https://dnjsrmstm1:<PAT>@github.com/dnjsrmstm1/DpsGameV2.git"
```
- git remote: `origin`=구 DpsGame, `neworigin`=DpsGameV2
- **GitHub Pages 설정 권장**: Settings→Pages→Source = `Deploy from a branch` + `gh-pages` /(root) → Action 큐 안 거치고 즉시 호스팅

## 이번 세션 주요 변경사항

### 보석 (강화/효과)
- 강화 보석 라벨: 하급/중급/상급/특급/고급/궁극/초월 "보석"
- max 변경: 재물 200, 경험 1000, 보호 200, 수호 1000, 초월 350, 인내 1000
- 섬세/채굴 보석 삭제
- **궁극 보석**: 44~47강 → **1~47강 전체 +0.01%/개**
- 비용 동적: `base × (1+보유/10)²` (quadratic)
- 재물 보석 base 1 → 100 (비용 상향)
- %배수 효과 전부 복리(1.X^N): 재물·경험·채광·자동화 등
- 자동화 보석 → 자동구입 배수 자동 동기화 (min(10, 1+보석수))

### 보주 (24종, xlsx 기반)
- 하급/중급/상급/특급/고급/경험/절약/힘/지위/권력/세공/신위/절명/풍요/폭식/영겁/명운/성운/천운/무위/풍성/부유/무엄/공명
- 응무조 비용·Max·효과설명 전부 반영
- 강화시도 단계별 보너스 적용 (54~59강): 하급/영겁→54, 중급/영겁→55, 상급/명운→56융합, 특급/성운→57, 고급/천운→58, 무위/무엄→59
- **미적용(광산 시스템 필요)**: 풍요/풍성/부유/세공/절약, 공명(레벨XP), 경험(초월경험), 신위(기본공격력)

### 크리스탈
- 제작 시스템: 노말 500조각 / 레어 1만 / 유니크 100만 / 갤럭시 3000만 / **퀘이사 = 상급조각 10개**
- 하락방어(개별확률 감소) 효과 전부 제거
- 무색/판매 배수 복리화 (총명 ×1.5, 흑색 ×2, 우주 ×3 등)

### 강화 / 레벨
- **다음경험치 = 10×lv** (선형, 대폭 감소). 캐릭레벨 최대 30만
- **30만 레벨 도달 → 초월레벨 1 자동 활성** (유일한 초월 unlock 경로)
- **50→51 초월**: base 0.5% + 누적(초월스텟/인내보석/명칭). 51강 도달은 초월시스템과 무관
- **56강 융합**: 55강 마린 20000마리 소모(절약보주 1개당 -10, min 5000) + 본인 강화. tick당 1회
- 강화쿨다운 800→300ms
- 강화 최초 달성 보상(45~50강): 크레딧+무색 1회 (45:5M+32 … 50:10B+3.2M)

### 보스 / 타격수
- 보스 **10단계** (시작 12→8), 보스당 사냥터+4, 공격력 ×1.5/단계(max ×6), 캐릭레벨+500
- **ExtraBoss 완전 제거** → 10보스 클리어 후 보스존 종료(텅 빔)
- 타격수 마일스톤표 갱신 (15000~1자): ExP/조각/박스/은하 + 크레딧/상급조각 보상
- 타격수 🎯 클릭 → 다음 보상 안내
- **51강+ 데미지 = 타격수** (이하 +1)

### 재화 / 경제
- **크레딧 = 유닛 판매 비용** (강도별: 41~44=100 … 50=1억 … 60=1양). 부족 시 판매 취소 + "크레딧 부족" 메시지
- **크레딧 마이너스 방지** (판매소 진입 시 검증)
- **상급크리스탈조각** 신규 재화 (타격수 보상 100조+/50경에서 획득, 퀘이사 제작용)
- 환생 보상: 크레딧 → **ExPoint** 로 변경
- **ExPoint→크레딧 교환** 버튼 (재화탭, 1ExP=10만 크레딧)
- 사냥터3 = 광산 (크레딧 dmg/1000 + 돈 동시)
- 사냥터 단가 10x 감소 (Lv2=1만, Lv3=10억)

### 사냥터 / 유닛
- **사냥터 1+2+3 합산 cap 48** (8 + 보스×4)
- 고유유닛 **사냥터1 고정** (자동 라우팅 제거)
- 고유유닛 클릭 시 강화창 안 열림 (선택만), 강화는 🦸 버튼으로
- 고유유닛 강화 MAX 버튼 추가 (크레딧 닿는 만큼)
- 32강 이상 유닛 구입 추가 (34~46강, 비용 사용자 제공)
- **자동구입 배수 1~10** 화살표 (자동패널)
- 이동속도 160→280 (cap 700)
